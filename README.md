# Inward App

## Requirements

* NodeJS
* NPM

## Installation

Running 'npm install' installs the following dependencies:

* body-parser
* ejs
* express

## Running the App

From the root directory of Inward-App, run `node app.js`. Visit the localhost address as linked from the terminal.
